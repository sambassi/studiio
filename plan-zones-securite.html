1. Architecture dans le code

⚙️ Logique d'affichage

État React

const [safeZonePlatform, setSafeZonePlatform] = useState<string | null>(null)

null = aucun guide affiché. 'instagram' | 'tiktok' | 'youtube' | 'facebook'

Quand les guides apparaissent

  • • L'utilisateur clique une icône plateforme → les zones s'affichent
  • • L'utilisateur commence un drag → les zones s'affichent aussi (si plateforme sélectionnée)
  • • Re-clic sur la même plateforme → les zones disparaissent (toggle)

Ce qui ne change PAS

  • • video-composer.ts — AUCUNE modification. Les zones sont un overlay CSS pur.
  • • Export vidéo — Les zones ne sont JAMAIS dans le rendu final.
  • • calendar/page.tsx — Pas impacté. Les guides sont uniquement dans l'infographie.

💡 Conseil clé : TikTok est la plateforme la plus restrictive (boutons plus hauts, description plus large). Si le contenu est lisible avec le guide TikTok activé, il sera lisible partout.
